El proyecto es la "Extención de la tarjeta HCAL" que se trata de una brindar nuevos perifericos a de la tarjeta principal HCAL basada en una FPGA para darle nuevas funcionalidades como; conversión ADC, comunicación RS232, comunicación USB, interfaz simple de usuario (LEDs y pulsadores), 4 salidas de PWM a 5V e interfaz para manejo de Relays a 12V. 

Se adjunta 

En la carpeta DOC las reglas proporcionadas por el fabricador local.
Carpeta de hojas de datos de los componentes principales (datasheets)
Carpeta con la lista de componentes (BOM)
Carpeta con los Gerbers para fabricación
Carpeta con los modelos de los componentes (lib.pretty)
Carpeta con los modelos 3D del proyecto (shapes3D)
